# MMD
前端markdown解析插件(js)，技术有限，只会实现几个常用的md功能

## 开发进度
### 2018.6.15
开始第二版的设计，前一版的设计实在不堪入目，干脆重新写一版算了，
尽量降低代码的耦合程度，也要提高代码的服用度。

## 开发目标
- 标题
- 图片
- 链接
- 有序/无序列表
- 代码块
- 引用块
- 表格
